# RestaurantAndHotel_bot - 幫你推薦在地美食的機器人

## **背景**
你是否開開心心規劃要到外縣市遊玩時，雖然有大概查過資料，但還是有些隱藏的在地小吃想要嘗試，卻不知道要從何下手?
即使聽過該餐廳，但對該店的評價及住址還是有所不確定，又不想花時間看網路上大量的評價是不是適合自己。
只要在Discord上加入RestaurantAndHotel_bot，就能以自然語言找尋你要的餐廳的資訊喔。
